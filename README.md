## 🚀 What This App Does

Codesila-devopsUI is a web app for managing DevOps work in one place. You can use it to track CI/CD pipelines, deployments, incidents, runbooks, and team work.

It is built for people who want a clear way to handle software release tasks without switching between many tools. The app runs with React, Node.js, PostgreSQL, and Docker.
